Trabajo 1 - Introducción a la Programación Numérica Programación Numérica

La programación numérica es una rama de la programación que busca resolver problemas matemáticos mediante aproximaciones, usando algoritmos que permiten trabajar con cálculos complejos cuando no es posible obtener una solución exacta. Se basa en métodos como integración, derivación, optimización o resolución de ecuaciones, y tiene como objetivo dar resultados confiables de forma eficiente con la ayuda de la computadora.

En cuanto a sus aplicaciones, en Ingeniería Estadística se emplea para estimar parámetros de modelos a través de algoritmos de optimización y también en simulaciones Monte Carlo para analizar la incertidumbre. En Ingeniería Informática, se utiliza en el entrenamiento de modelos de inteligencia artificial y machine learning mediante métodos numéricos como el descenso de gradiente, así como en el área de gráficos y visión por computadora para realizar interpolaciones, transformaciones y simulaciones visuales.

Actividad 2: Ejemplo en Python
Programa: f(x,y) = 2*y + x

Este programa pide dos números, calcula r = 2*y y finalmente f(x,y) = 2*y + x:


import re

NUM_REGEX = r'^[+-]?(\d+(\.\d*)?|\.\d+)$'

def pedir_numero(nombre):
    while True:
        s = input(f"Ingrese {nombre}: ").strip()
        if re.match(NUM_REGEX, s):
            return float(s)
        print("Entrada inválida. Introduce un número (ej: 3, -2, 4.5, -0.7)")

def f(x, y):
    return 2 * y + x

def main():
    print("Programa: calcula r = 2*y y resultado = r + x usando f(x,y) = 2*y + x")
    x = pedir_numero("x")
    y = pedir_numero("y")

    r = 2 * y
    resultado = r + x
    resultado_func = f(x, y)

    print("\nValores calculados:")
    print(f"  x = {x}")
    print(f"  y = {y}")
    print(f"  r = 2 * y = {r}")
    print(f"  r + x = {resultado}")
    print(f"  f(x, y) = 2*y + x = {resultado_func}")

if __name__ == "__main__":
    main()
Actividad 3: Graficador ASCII
Programa: Graficador ASCII de funciones

Este programa grafica funciones matemáticas en un plano cartesiano usando caracteres ASCII:


import math
import re

def preparar_funcion(expr):
    # Quitar espacios
    expr = expr.replace(" ", "")
    # Insertar * donde falte (ej: 5x -> 5*x)
    expr = re.sub(r"(\d)([a-zA-Z])", r"\1*\2", expr)
    return expr

def graficar(funciones):
    ancho = 80   # columnas (eje X)
    alto = 25    # filas (eje Y)
    escala = 5   # cuántas unidades por división

    # Matriz vacía
    plano = [[" " for _ in range(ancho)] for _ in range(alto)]

    # Dibujar ejes
    for x in range(ancho):
        plano[alto//2][x] = "-"  # eje X
    for y in range(alto):
        plano[y][ancho//2] = "|"  # eje Y
    plano[alto//2][ancho//2] = "+"  # origen

    # Colores simulados con símbolos diferentes
    simbolos = ["*", "#"]

    for idx, (funcion, simbolo) in enumerate(zip(funciones, simbolos)):
        if funcion.strip() == "":
            continue

        funcion = preparar_funcion(funcion)

        for col in range(ancho):
            # convertir pixel en valor x real
            x = (col - ancho//2) / escala
            try:
                y = eval(funcion, {"x": x, "math": math})
                fila = int(alto//2 - y * escala)
                if 0 <= fila < alto:
                    plano[fila][col] = simbolo
            except Exception as e:
                print("Error en función:", funcion, e)

    # Imprimir
    for fila in plano:
        print("".join(fila))

# Ejemplo
funciones = ["2*x+5", "x**2/5"]
graficar(funciones)
