üåü TEMA: M√âTODOS NUM√âRICOS PARA ENCONTRAR RA√çCES

Definici√≥n: Se utilizan m√©todos num√©ricos para encontrar los valores de x que satisfacen f(x)=0, especialmente cuando no es posible resolver la ecuaci√≥n de manera algebraica. Estos m√©todos buscan aproximar la ra√≠z con la precisi√≥n deseada mediante iteraciones.

üîπ Clasificaci√≥n de M√©todos

M√©todos Cerrados (de intervalo):

Requieren un intervalo [a, b] donde la funci√≥n cambia de signo (f(a)f(b)<0). Garantizan la convergencia.

  • M√©todo de Bisecci√≥n: Divide el intervalo a la mitad y selecciona el subintervalo donde la funci√≥n cambia de signo. Se repite hasta alcanzar la precisi√≥n deseada. Ventaja: siempre converge. Desventaja: lento.
  • M√©todo de Regla Falsa (Falsa Posici√≥n): Aproxima la ra√≠z usando la l√≠nea recta que une los extremos del intervalo. M√°s r√°pido que la bisecci√≥n en muchos casos.

M√©todos Abiertos (sin necesidad de intervalo):

  • M√©todo de Newton-Raphson: F√≥rmula: x‚Çô‚Çä‚ÇÅ = x‚Çô - f(x‚Çô)/f'(x‚Çô). Requiere derivada de la funci√≥n. Ventaja: muy r√°pido si la aproximaci√≥n inicial es buena. Desventaja: puede fallar si la derivada es cero o la aproximaci√≥n inicial est√° lejos de la ra√≠z.
  • M√©todo de la Secante: Similar a Newton-Raphson pero no requiere la derivada; usa dos aproximaciones iniciales. F√≥rmula: x‚Çô‚Çä‚ÇÅ = x‚Çô - f(x‚Çô) * (x‚Çô - x‚Çô‚Çã‚ÇÅ)/(f(x‚Çô) - f(x‚Çô‚Çã‚ÇÅ)).
  • M√©todo de Punto Fijo: Se transforma f(x)=0 en x=g(x) y se usa x‚Çô‚Çä‚ÇÅ = g(x‚Çô). Converge si |g'(x)| < 1 cerca de la ra√≠z.
üîπ Ejemplo de Newton-Raphson en Python

# M√©todo de Newton-Raphson para encontrar ra√≠ces de f(x) = 0

def f(x):
    return x**3 + x - 1  # Funci√≥n a evaluar

def df(x):
    return 3*x**2 + 1    # Derivada de la funci√≥n

def newton_raphson(x0, tol=1e-6, max_iter=100):
    xn = x0
    for n in range(max_iter):
        fxn = f(xn)
        dfxn = df(xn)
        if dfxn == 0:
            print("Derivada cero. No se puede continuar.")
            return None
        xn1 = xn - fxn/dfxn
        print(f"Iteraci√≥n {n+1}: x = {xn1}, f(x) = {f(xn1)}")
        if abs(xn1 - xn) < tol:
            print(f"\nLa ra√≠z aproximada es: {xn1}")
            return xn1
        xn = xn1
    print("No se alcanz√≥ la tolerancia requerida")
    return None

# Aproximaci√≥n inicial
x0 = 0.5

# Ejecutar el m√©todo
newton_raphson(x0)
