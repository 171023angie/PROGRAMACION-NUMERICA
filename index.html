TEMA: VARIABLES Y FUNCIONES

SUBTEMA 🔹 ¿Qué es una variable?

Una variable es un símbolo o elemento que puede cambiar de valor dentro de un problema o sistema. En estadística, una variable representa una característica que puede medirse u observarse y que varía entre individuos o elementos de una población. Ejemplo: edad, salario, temperatura, número de fallas, tiempo de ejecución de un programa. En informática, una variable es un espacio de memoria que guarda un valor que puede cambiar durante la ejecución de un programa.

SUBTEMA 🔹 ¿Qué es una función?

Una función es una relación entre variables, donde a cada valor de la variable independiente (x) le corresponde un solo valor de la variable dependiente (y). En estadística e ingeniería, una función se usa para modelar relaciones entre fenómenos. Ejemplo: y = 2x + 3 → si x = 5, entonces y = 13. En informática, una función es un bloque de código reutilizable que realiza una tarea específica y puede recibir datos (parámetros) y retornar un resultado.

TEMA: RESTRICCIONES

Las restricciones son condiciones o limitaciones que debe cumplir un problema matemático o de programación lineal, delimitando la región factible.

SUBTEMA 🔹 1.1 Tipos de Restricciones

a) Restricciones de Igualdad: Son condiciones que deben cumplirse exactamente.

Ejemplo: x + y = 10

b) Restricciones de Desigualdad: Permiten límites máximos o mínimos.

Ejemplo: x + y ≤ 10

c) Restricciones de Límites: Definen un rango permitido para cada variable.

Ejemplo: 0 ≤ x ≤ 5

import math
import re

def preparar_funcion(expr):
    # Quitar espacios
    expr = expr.replace(" ", "")
    # Insertar * donde falte (ej: 5x -> 5*x)
    expr = re.sub(r"(\d)([a-zA-Z])", r"\1*\2", expr)
    return expr

def graficar_restricciones(restricciones, fronteras, ancho=80, alto=25, escala=2):
    # Matriz vacía
    plano = [[" " for _ in range(ancho)] for _ in range(alto)]

    # Dibujar ejes
    for x in range(ancho):
        plano[alto//2][x] = "-"  # eje X
    for y in range(alto):
        plano[y][ancho//2] = "|"  # eje Y
    plano[alto//2][ancho//2] = "+"  # origen

    # Sombrar región factible
    for col in range(ancho):
        for fila in range(alto):
            x = (col - ancho//2) / escala
            y = (alto//2 - fila) / escala
            try:
                if all(eval(cond, {"x":x,"y":y,"math":math}) for cond in restricciones):
                    plano[fila][col] = "."
            except Exception:
                pass

    # Dibujar fronteras (rectas o verticales)
    simbolos = ["*", "#", "+", "x"]
    for idx, expr in enumerate(fronteras):
        expr = preparar_funcion(expr)
        simbolo = simbolos[idx % len(simbolos)]
        for col in range(ancho):
            x = (col - ancho//2) / escala
            try:
                y = eval(expr, {"x": x, "math": math})
                fila = int(alto//2 - y * escala)
                if 0 <= fila < alto:
                    plano[fila][col] = simbolo
            except Exception:
                # Interpretar como vertical x = constante
                try:
                    vx = float(expr)
                    col_v = int(round(vx * escala + ancho//2))
                    if 0 <= col_v < ancho:
                        for fila_v in range(alto):
                            plano[fila_v][col_v] = simbolo
                except Exception:
                    pass

    # Imprimir plano
    for fila in plano:
        print("".join(fila))

TEMA: MÉTODOS NUMÉRICOS PARA ENCONTRAR RAÍCES

Definición: Se utilizan métodos numéricos para encontrar los valores de x que satisfacen f(x)=0, especialmente cuando no es posible resolver la ecuación de manera algebraica. Estos métodos buscan aproximar la raíz con la precisión deseada mediante iteraciones.

SUBTEMA 🔹 Método de Newton-Raphson

Fórmula: xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ).
Requiere derivada de la función.
Ventaja: muy rápido si la aproximación inicial es buena.
Desventaja: puede fallar si la derivada es cero o la aproximación inicial está lejos de la raíz.

ACTIVIDAD: CÓDIGO

def f(x):
    return x**3 + x - 1  # Función a evaluar

def df(x):
    return 3*x**2 + 1    # Derivada de la función

def newton_raphson(x0, tol=1e-6, max_iter=100):
    xn = x0
    for n in range(max_iter):
        fxn = f(xn)
        dfxn = df(xn)
        if dfxn == 0:
            print("Derivada cero. No se puede continuar.")
            return None
        xn1 = xn - fxn/dfxn
        print(f"Iteración {n+1}: x = {xn1}, f(x) = {f(xn1)}")
        if abs(xn1 - xn) < tol:
            print(f"\nLa raíz aproximada es: {xn1}")
            return xn1
        xn = xn1
    print("No se alcanzó la tolerancia requerida")
    return None

# Aproximación inicial
x0 = 0.5

# Ejecutar el método
newton_raphson(x0)
